html基本结构

```
<!doctype html>	<!-- 声明文档类型 html5版本 具有向下兼容低版本的功能  -->
<html>
   <head><!--头部 : 不可视区域：看不到的内容 -->
      <meta charset='utf-8'/>	<!-- 编码格式：万国码 -->
      <title> 九万字 </title>	<!-- 网页的三要素：标题 ，关键词 -->
      <meta name='keywords' content='帅，除了帅，还是帅！'/>	<!-- 元标签 -->
      <meta name='description' content='浮生未歇'/>
   </head>
   <body><!-- 身体: 可视区域。 -->

      <!--
         什么是标签 
            标签是由一对尖括号组成的。
         种类：
            单标签：<img />
            双标签：<html></html>
               双标签是由同一对标签组成的，前面<html>是开始标签，后面</html>是结束标签。
      -->

   </body>
</html>
```

常用标签

```
标题标签：
   h1 一级标题
   h2 二级标题
   h3 三级标题
   h4 四级标题
   h5 五级标题
   h6 六级标题
特点：
   字体都是有加粗的效果同时h1到h6的字体逐渐变小！
   与seo息息相关！
   
	p  段落标签
		strong/b标签 ：加粗标签
		两者区别在前者与seo相关，后者只是普通加粗的效果！
		
	em/i  倾斜的效果
		em与seo相关，i标签只是普通倾斜的效果。
		
	span  区分样式的标签
		与普通的文本样式区别开来
		
	sup  上标
	sub  下标	
```

相对路径绝对路径

```

   相对路径：
      相对于.html文本出来的。
   绝对路径（）：
      两种绝对路径：
         从盘符出发
            
         从域名出发

缺点：
   绝对路径的缺点在于，如果当时的路径发生改变，图片就会消失。
```

table

```
table:定义标签的表格
    table特殊情况：border='num'是自带的属性！
表格简化内容：可以不加thead,tbody,tfoot,但是默认把所有的行与列都放到tbody里面去。   
样式问题：
    th,td自带一个像素的内边距

表格的特征：
    1.每一个单元格会平分table的宽度；    
    2.table决定整个表格的宽度；
    3.th标签自带默认的加粗以前水平居中，垂直居中的属性；
    4.td标签自带默认的水平居左，垂直居中的属性；
    5.表格文字样式多的情况下，分到的比较也就比较多；
    6.table表格支持margin,padding；padding会受边框合并的影响；
    7.td支持padding但不支持margin;

合并表格：
    colspan 合并列
    rowspan 合并行
```
